This Chinese pinyin input method is mainly for China users.
The supported input language is Simplify Chinese (zh_CN) and Mandarin.

Main features:

+ Chinese pinyin input
+ Number / symbol / letter input
+ Learning / custom input
+ No-log mode
+ Run in local absolutely (No network permission)
+ Clipboard manager
+ and more !

Supported architecture: arm64-v8a

Full document: https://github.com/fm-elpac/a_pinyin
